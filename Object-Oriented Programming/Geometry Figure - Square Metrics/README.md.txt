# Geometry Figure - Square Metrics

## Description
This program calculates the perimeter, area, and diagonal of a square using the coordinates of its four vertices. It also checks if the given points form a square.

## Features
- Calculates perimeter, area, and diagonal of a square.
- Checks if the points form a valid square.

## How It Works
1. The user inputs the coordinates of four points.
2. The program checks if they form a square and calculates the metrics if true.

## Sample Input
Enter coordinates for point 1 (x1, y1): 0 0 
Enter coordinates for point 2 (x2, y2): 4 0 
Enter coordinates for point 3 (x3, y3): 4 4 
Enter coordinates for point 4 (x4, y4): 0 4

or:

Enter coordinates for point 1 (x1, y1): 0 0 
Enter coordinates for point 2 (x2, y2): 0 2 
Enter coordinates for point 3 (x3, y3): 2 2 
Enter coordinates for point 4 (x4, y4): 2 0
